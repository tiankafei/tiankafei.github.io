# 数仓分层

  1. ODS层：原始数据层，存放原始数据，直接加载原始日志、数据，数据保持原貌不做处理
  2. DWD层：对ODS层数据进行清洗（去除空值，脏数据，超过极限范围的数据）、维度退化脱敏等
  3. DWS层：以DWD为基础，按天进行轻度汇总。
  4. DWT层：以DWS为基础，按主题进行汇总
  5. ADS层：为各种报表提供数据
